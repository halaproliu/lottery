### 项目初始化
```js
npm i 
或
yarn
```

### 项目启动
```js
npm start
或
yarn start
```

### 项目编译
```js
npm run build
或
yarn run build
```

### 框架依赖中间件
- koa-bodyparser（数据转换，支持json,form,text）
- koa-logger（日志）
- koa-static（配置静态目录）
- koa-cors（解决跨域问题）
- koa-convert（将koa1.0中间件转化为koa2.0中间件）
- koa-router（路由）

### 使用ES6 Decorator
依赖@babel/plugin-proposal-decorators(babel7)插件实现decorator定义接口路由
